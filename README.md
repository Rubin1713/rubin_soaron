# rubin_soaron
New codes for UI backend integration
    HM30_batt_lvl.py - HM30 battery level ( ground unit ) to output console in percentage
    TypeC_telem_NonUI.py - Telemetry data from typeC port to output console not on UI.
    TypeCtelem.py - Telemetry data from typeC port to UI.
    udptelem.py - Telemetry data from LAN port to UI.
    udp_nonUI.py - Telemetry data from LAN port to output console not on UI.
    surface_recognition.py - Surface recognition by hm30 camera and identify type of surface.
    auto_record.py - Record the video of drone and stored it in Videos folder.
    WifiBluetooth.py - Read the Wifi and bluetooth signal of Remote control and printit on output console.
